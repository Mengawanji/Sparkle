# Plumbor - Plumbing Services React Template

Plumbor is a modern, responsive, and professionally designed React + TypeScript template built for plumbing services, maintenance companies, repair businesses, handymen, home service providers, and related service industries.

The template is developed using React, TypeScript, Vite, Redux Toolkit, and SCSS, ensuring excellent performance, maintainability, and scalability.

---

## Features

* React 19
* TypeScript
* Vite Build Tool
* Redux Toolkit State Management
* React Router DOM
* Fully Responsive Design
* Modern and Clean User Interface
* Service Pages
* Project Pages
* Team Pages
* Blog Pages
* Shop Pages
* Contact Forms
* Booking Forms
* Smooth Animations
* SCSS Support
* Cross Browser Compatible
* Easy Customization

---

## Requirements

* Node.js 18 or Higher
* npm 9 or Higher

---

## Installation

Install all dependencies:

```bash
npm install
```

---

## Development

Run the development server:

```bash
npm run dev
```

The project will start on:

```text
http://localhost:5173
```

---

## Production Build

Create production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Folder Description

| Folder         | Description                     |
| -------------- | ------------------------------- |
| api            | API services and configurations |
| components     | Reusable React components       |
